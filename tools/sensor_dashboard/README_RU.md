# STM32 Sensor Dashboard

Графическая панель для отображения данных от датчиков:

- MPU6050 — акселерометр, гироскоп и внутренняя температура;
- MLX90614 — температура самого ИК-датчика и бесконтактная температура объекта.

Данные поступают от STM32 через USB-UART со скоростью `115200 baud`.

## Возможности

На главном экране отображаются:

- температура объекта в °C;
- температура MLX90614 в °C;
- температура MPU6050 в °C;
- наклон платы вокруг осей X и Y в градусах;
- суммарное ускорение в `g`;
- суммарная угловая скорость в `°/с`;
- состояние движения;
- разница температур объекта и MLX90614;
- статус I²C и подключённых датчиков;
- графики в реальном времени;
- запись результатов в CSV.

Исходные единицы STM32:

- `mg`;
- `mdps`;
- `cC`.

На главном экране они автоматически преобразуются в понятные единицы:

- `g`;
- `°/с`;
- `°C`;
- градусы наклона.

Сырые сообщения UART доступны только во вкладке **«Диагностика»**.

---

## Требования

Поддерживается Ubuntu и другие Debian-подобные Linux-системы.

Необходимы:

- Python 3;
- модуль Python `venv`;
- `pip`;
- доступ пользователя к последовательному порту;
- подключённая по USB плата STM32 NUCLEO.
